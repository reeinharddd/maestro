Usage Guidelines

Primary lockup

Icon + wordmark on dark bg. Use on landing page, docs header, splash screen. Never recolor or distort.

Icon standalone

App icon, avatar, CLI --version output, favicon. Use full-color on dark, mono on light.

Mono icon

Small sizes (<32 px), print, watermarks, constrained palettes. One stroke weight, no gradient.

ASCII art

CLI splash, README.md, --help header, MOTD. Use compact for terminals <60 cols.

Favicon

SVG in <link rel="icon" type="image/svg+xml">. Gold #c8a064 on dark, #8a6c3a on light via media queries.

Do not

Rotate the baton. Change the angle. Recolor to non-gold. Outline in white. Add text inside icon.
